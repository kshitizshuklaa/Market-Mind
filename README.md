## 📖 Overview

**Market Mind** is a high-performance, production-grade financial analytics dashboard. It is designed to provide real-time market tracking, advanced technical analysis, and neural-driven forecasting—all within a stunning "Quantum" aesthetic. 

Unlike traditional apps that rely on expensive APIs, Market Mind uses **Local Intelligence** to process data directly in your browser, ensuring privacy, speed, and zero cost.

---

## ✨ Key Features

### 📈 Real-Time Analytics
- **Live Market Tracking:** Real-time price simulations for **50+ global stocks**, including tech giants, crypto, and indices.
- **Technical Indicators:** Instant calculation of **RSI (14)**, **MACD**, **SMA**, **EMA**, and **Volatility** metrics.
- **Interactive Charts:** High-fidelity, responsive charts with multi-timeframe support.

### 🧠 Neural Intelligence (Local)
- **Trend Forecasting:** Uses **Linear Regression** for short-term price direction.
- **Pattern Recognition:** Implements a **Random Forest Regressor** to identify non-linear market patterns.
- **Sentiment Analysis:** Neural analysis of global news to determine market mood.

### 🎨 Premium User Experience
- **Quantum Aesthetic:** A dark-themed, glassmorphic UI with fluid neural animations.
- **Asset Comparison:** Compare up to 3 assets side-by-side with performance alignment.
- **Global Intelligence:** Curated news feed with sentiment impact scoring.
- **Fully Responsive:** Seamless experience across Mobile, Tablet, and Desktop.

## 🚀 Getting Started (Local Setup)

Follow these steps to run Market Mind on your personal computer:

1. **Clone the Project:**
   ```bash
   # Open your terminal and navigate to your project folder
   cd your-project-folder
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   Navigate to `http://localhost:3000` to see your app live!

## 📂 Core Project Structure

To keep your project clean, here are the **Essential Files** you need:

- `src/App.tsx`: **The Heart.** Contains all UI components, state logic, and dashboard layout.
- `src/lib/ml.ts`: **The Brain.** Contains the local Machine Learning models for forecasting.
- `src/lib/math.ts`: **The Calculator.** Handles all technical indicator calculations (RSI, MACD, etc.).
- `src/constants.ts`: **The Data.** Stores the list of 50+ stocks and global configuration.
- `src/index.css`: **The Style.** Contains the Tailwind CSS configuration and global theme.
- `package.json`: **The Blueprint.** Lists all the libraries required to run the app.
